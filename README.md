# Aaegis Token Metadata

Public metadata for the **AGSUSD** stablecoin of the Aaegis Protocol.

This repository stores the token logo and metadata JSON used by Solana wallets, explorers, and dApps.

## Files

- **token-metadata.json**  
  Metadata file following the Metaplex Token Metadata standard.  
  Contains the token name, symbol, description, and logo reference.

- **aaegis-protocol.png**  
  Token logo (PNG format) used by wallets such as Phantom, Solflare, and Backpack.

## Usage

Set the metadata URI in your on-chain Token Metadata to:

```

[https://raw.githubusercontent.com/shivanshu814/aaegis-token-metadata/main/token-metadata.json](https://raw.githubusercontent.com/shivanshu814/aaegis-token-metadata/main/token-metadata.json)

```

Wallets will automatically read this file to display:
- Token name  
- Symbol  
- Logo  

## About

This repository is intended to provide a permanent public source for Aaegis Protocol's token metadata.
